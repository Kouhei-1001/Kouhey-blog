JavaScriptのforEachの使い方

2024年3月15日 JavaScript

forEachとは

forEachは、JavaScriptの配列メソッドの1つで、配列の各要素に対して順番に処理を実行するために使用されます。for文やwhile文を使わずに、より簡潔に配列の要素を処理できる便利なメソッドです。

基本的な使い方

forEachの基本的な構文は以下のようになります：


const numbers = [1, 2, 3, 4, 5];

numbers.forEach((number) => {
  console.log(number);
});

引数の説明

forEachメソッドは、コールバック関数を引数として受け取ります。このコールバック関数には以下の3つの引数を指定できます：

  • 現在の要素の値
  • 現在の要素のインデックス
  • 処理対象の配列

const fruits = ['りんご', 'バナナ', 'オレンジ'];

fruits.forEach((fruit, index, array) => {
  console.log(`${index + 1}番目の${fruit}`);
  console.log('配列全体:', array);
});

実践的な使用例

DOM要素の操作


const buttons = document.querySelectorAll('button');

buttons.forEach((button) => {
  button.addEventListener('click', () => {
    console.log('ボタンがクリックされました');
  });
});

オブジェクトの配列を処理


const users = [
  { name: '田中', age: 25 },
  { name: '佐藤', age: 30 },
  { name: '鈴木', age: 28 }
];

users.forEach((user) => {
  console.log(`${user.name}さんは${user.age}歳です`);
});

注意点

  • forEachは配列の要素を変更できません（非破壊的メソッド）
  • breakやcontinueのような制御文は使用できません
  • 非同期処理には適していません
